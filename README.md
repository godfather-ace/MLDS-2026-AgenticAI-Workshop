# MLDS-2026-AgenticAI-Workshop

This repository contains the presentation slides, source code, and deployment manifests for two sessions delivered by Sachin Tripathi

---

## 🛠️ Workshop 1: From Prototype to Production: Battle-Tested Agentic AI

This session offers a technical deep dive into building and deploying agentic applications at scale on AWS. Using AWS Copilot, we will demonstrate how to automate infrastructure provisioning and CI/CD pipelines, freeing developers to focus on core agent logic. The workshop will also cover how to integrate advanced components like the Model Context Protocol (MCP), a key standard that enables agents to access external tools and data sources, which is essential for creating powerful, context-aware applications that can move beyond simple chat and perform real-world tasks.

### 🎯 Key Takeaways
* Learn how to create AI agents that can handle real-world tasks and deploy them in a production environment.
* Understand how to use tools like AWS Copilot to automate the provisioning of infrastructure and set up CI/CD pipelines.
* Gain knowledge of how MCP enables agents to access and use external data sources and tools.
* Build and deploy reliable and scalable real-world agentic solutions.

### 📋 Prerequisites
* Proficiency in Python programming.
* Familiarity with AI/GenAI concepts.
* Prior experience with AWS cloud services is beneficial.
* Familiarity with Docker and containerization is helpful for deployment aspects.

---

## ⚔️ Workshop 2: Protocol Wars: MCP, A2A, and the Battle for Agent Interoperability

Explore the emerging standards that will define how agents communicate and collaborate. This workshop provides hands-on experience with Model Context Protocol (MCP) for standardized context sharing, Agent-to-Agent (A2A) protocols for autonomous communication, and Agent Communication Protocol (ACP) implementations. Learn to build agents that speak multiple languages, compare the strengths and trade-offs of each protocol, and future-proof your systems as the ecosystem consolidates around winning standards.

### 🎯 Key Takeaways
* Classify and describe the distinct roles of MCP, A2A, and ACP protocols.
* Appraise and implement interoperable agents that can communicate using protocols.
* Understand and distill the strengths and tradeoffs of each protocol based on different use-cases and implementations.
* Learn and illustrate the design of adaptable and evolving agentic systems.

### 📋 Prerequisites
* Familiarity with AI agents and their working.
* Experience with agentic frameworks such as Google’s ADK, OpenAI Agents SDK, Autogen, CrewAI, etc.
* Understanding of communication protocols such as HTTP and REST is beneficial.
* Experience with API development is preferred but not mandatory.
